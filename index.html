  • Crossdevice carousel for jQuery

    Supports navigation via touch, click, and keyboard. Uses CSS3 animations when possible for smooth animations. Supports options including pagination, next/prev, page change callback, pagechange speed, keyboard nav, and tap to return

  • Initialize

    $('#carousel-container').responsiveCarousel();

    Customize

    var carousel = $('#wrapper').responsiveCarousel({
        direction: 'horizontal',
        transitionSpeed: 500,
        keyControl: true,
        arrows: true,
        pagination: true,
        paginationEl: 'paginationList',
        tapToReturn: true,
        callback: function() {
            console.log(this.state);
        }
    });
