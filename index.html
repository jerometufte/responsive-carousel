  • Crossdevice carousel for jQuery

    Enables carousel navigation via touch, click, and keyboard. Uses CSS3 animations when possible for smooth animations. Supports options including pagination, next/prev, page change callback, pagechange speed, keyboard nav, and tap to return. Can be used multiples on page as a widget, or as the centerpiece of a single-page website (like this one!).
